# Aion2 Overlay

게임 위에 띄워서 사용할 수 있는 `WinForms + WebView2` 기반 오버레이입니다.

## 기능

- `웹 / 아툹검색 / 침식 막넴 만두" 탑
- `항상 위`
- `클릭 통과`
- `투명도 조쥈`
- `자유 리사이즈`
- `업데이트` 버튼과 GitHub 릴리즈 확인

## 실행

```powershell
dotnet restore
dotnet build
dotnet run
```

## 요구 사항

- `.NET 8 SDK`
- `Microsoft Edge WebView2 Runtime`

## 실행 파일

디버그 빌드 기본 경로:

```text
bin/Debug/net8.0-windows/Aion2Overlay.exe
```
